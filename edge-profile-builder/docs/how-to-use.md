# How To Use EDGE Profile Builder

## Option 1: Use the Custom GPT

1. Open EDGE Profile Builder.
2. Answer the questions step by step.
3. Push for specificity when prompted.
4. Save the final output as `learner-profile.md`.
5. Paste it into any AI tool before asking for serious help.

## Option 2: Use the Prompt Manually

1. Open `prompts/edge-profile-builder-prompt.md`.
2. Copy the prompt.
3. Paste it into ChatGPT, Claude, Gemini, Perplexity, or another AI tool.
4. Answer the questions.
5. Save the final output as `learner-profile.md`.

## How To Use Your Profile

Before asking AI for personalised help, paste:

```text
I am sharing my EDGE Profile below. Use this to understand my current environment, direction, gaps, and preferred engagement style. Do not give generic advice. Ask for missing context when needed. Push back on vague answers. Help me think clearly and take practical action.

[Paste learner-profile.md here]
```

Then ask your question.
